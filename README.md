# DMiME
医学医療用語変換辞書「DMiME」の再配布用・保守用レポジトリ

DMiMEはオープンライセンスの医学医療用語変換辞書ファイルです。
無料で、誰でも使える、学習効率を向上させる入力ツールを目指して作成されました。
DMiMEは日本医師会総合政策研究機構ORCAの開発するかな漢字変換用医療辞書のVer.0.3を土台として作成されています。
このかな漢字変換用医療辞書がGPL2ライセンスであることからGPL2を継承し、改変と再配布が可能なオープンライセンスの医療用語変換辞書となります。

## インストール法

最新版は [Releases](https://github.com/c-colloid/DMiME/releases/latest) から辞書ファイルをダウンロードしてください。使用している IME に応じて以下の 3 種類のアセットがあります。

### 1. Google 日本語入力 (Windows / Mac / Linux)

**ダウンロード**: `DMiME-<バージョン>.zip`

1. zip を解凍して `DMiME-<バージョン>.txt` を取り出す
2. 画面右下 (Mac は上部メニュー) の Google 日本語入力アイコンを右クリック → **「辞書ツール」** を開く
3. 左上の **「管理」** メニュー → **「新規辞書にインポート」** をクリック
4. 解凍した txt ファイルを選択
5. 辞書名を `DMiME` などにして保存

40,000 語以上の医学用語が変換候補に出るようになります。

### 2. Microsoft IME (Windows)

**ダウンロード**: `DMiME-<バージョン>-msime.zip`

1. zip を解凍して `DMiME-<バージョン>.txt` を取り出す (文字コードは UTF-16 LE BOM、改行は CRLF)
2. 画面右下の「あ」または「A」を右クリック → **「単語の登録」** → **「ユーザー辞書ツール」**
3. **「ツール」** メニュー → **「テキスト ファイルからの登録」**
4. 解凍した txt ファイルを選択するとインポート処理が始まる

> **補足**: Google 日本語入力用とは中身の形式が異なります (文字コード・改行・品詞体系)。また Google 日本語入力固有の「サジェストのみ」品詞を持つ 11 エントリは MS IME に概念がないため MS IME 版では除外されています (42,456 エントリ)。

### 3. Mac ユーザ辞書 (iCloud 同期)

**ダウンロード**: `DMiME-<バージョン> igakujisho for icloud.plist`

1. plist ファイルをダウンロード
2. ファイルをダブルクリックすると「システム設定 → キーボード → ユーザー辞書」が開き、インポートが始まる

Mac 標準 IME のユーザ辞書として登録されます。iCloud 経由で他の Apple デバイス (iPhone/iPad) とも自動同期されます。

## 開発者向け

辞書データは `src/` 以下の 16 ファイル (50音×清濁別 TSV) がマスタです。配布物は `python3 scripts/build_release.py` で再生成されます。編集ルールは [CONTRIBUTING.md](CONTRIBUTING.md) 参照。
